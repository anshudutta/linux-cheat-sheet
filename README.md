# Linux Cheat Sheet
## File System
* Navigation
```bash
cd /path/to/directory
# Go to root
cd /
# Go home
cd ~
# Print path
pwd
```
* List a directory
```bash
ls
# List vertically
ls -l
# List all
ls -la
# List any directory
ls /abs/path/dir
```
* Create a directory
```bash
mkdir somedir
```
* Delete a directory
```bash
rm -r somedir
```
* Create a file
```bash
touch somefile
```
* Delete a file
```bash
rm somefire
```
* Move a file
```bash
mv /path/to/source/somefile /path/to/dest/
```
